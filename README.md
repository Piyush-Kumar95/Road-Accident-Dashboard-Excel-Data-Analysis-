# Road Accident Dashboard (Excel Data Analysis) 
This project analyzes road accident data using Microsoft Excel to uncover patterns, trends, and insights related to accidents. The interactive dashboard helps in understanding accident severity, causes, and location-based risks.

# Objective 
- Analyze accident data to identify key trends
- Visualize accident patterns by time, location, and vehicle type
- Provide insights to improve road safety and decision-making

 # Tools & Technologies
 Microsoft Excel :
 - Pivot Tables
 - Pivot Charts
 - Slicers
 - Conditional Formatting
 - Data Cleaning

# Dashboard Features
- Total number of accidents overview
- Accident severity analysis (Fatal, Serious, Minor)
- Year-wise and month-wise trends
- Location-based accident distribution
- Vehicle type analysis
- Interactive filters using slicers

# Dataset Description
- Accident Date
- Location
- Vehicle Type
- Accident Severity
- Weather Conditions
- Road Type
- Number of Casualties

# Steps Performed
- Data Cleaning (removed duplicates, handled missing values)
- Data Transformation (formatted dates, categorized data)
- Created Pivot Tables for analysis
- Built interactive dashboard using charts and slicers
- Derived insights for decision-making
